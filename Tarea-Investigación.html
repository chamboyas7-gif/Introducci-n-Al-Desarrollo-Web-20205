Diferencias entre etiquetas HTML

<b> vs <strong>

Bold (b): Aplica negrita sin significado semántico.

Strong: Aplica negrita pero con énfasis, Google le da más importancia al texto dentro de <strong>.

<i> vs <em>

Italic (i): Aplica cursiva solo visualmente.

Emphasis (em): Aplica énfasis en cursiva y puede ser interpretado como más relevante por buscadores.

<s> vs <del>

Strikethrough (s): Tacha el texto solo visualmente.

Deleted (del): Indica que el texto fue eliminado, tiene valor semántico y puede ser usado por herramientas de lectura.

Elementos de línea y de bloque

¿Qué son?

Los elementos de línea no comienzan en una nueva línea y solo ocupan el espacio necesario. Se usan dentro de párrafos o bloques.
Los elementos de bloque comienzan en una nueva línea y ocupan todo el ancho disponible del contenedor.

5 ejemplos de elementos de línea:

  • <span>
  • <a>
  • <strong>
  • <em>
  • <img>

5 ejemplos de elementos de bloque:

  • <div>
  • <p>
  • <h1> a <h6>
  • <ul>
  • <form>

1. Input (type="text"):

El campo "input" en HTML crea un campo interactivo dentro de un formulario web para que los usuarios ingresen datos, como texto, contraseñas, números, o seleccionen opciones. Sus propiedades, controladas principalmente por el atributo type (que define el tipo de campo, como text, email, password, date, etc.), permiten especificar el formato, validación y comportamiento del campo
